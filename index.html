04/03/2024 - Mise à jour Beta V0.6

Pour cette version 0.6, l'expérience utilisateur a été complètement améliorée avec l'ajout de nouvelles fonctionnalités : page d'accueil avec les dernières informations, amélioration des éléments utilisateur, organisation des textes types, amélioration de la page éditeur.


La liste des principales nouvelles fonctionnalités :

    Page d'accueil
  • Encart mise à jour : informe les modifications apportées au site internet ;
  • Ajout d'une page d'accueil : permet de sélectioner ou ajouter un profil utilisateur avant de passer à la liste de message type.

    • Page bibliothèque
  • Ajout en-tête utilisateur : zone au dessus de la page où est affiché l'utilisateur actif, un bouton "Retour pour retourner à la page précédente et un bouton pour changer d'utilisateur ;
  • Classement des textes : les textes types ont été classés dans différents dossiers et sous-dossiers. Chaque dossier est associé à un public et un média, ou à un événement. Les sous-dossiers sont associés à une raison ;
  • Filtre par région : les textes affichés sont filtrés selon la région de l'utilisateur.

    • Page éditeur
  • Ajout d'un menu sur le côté : permet de changer de texte depuis l'éditeur ;
  • Ajout d'objet de mail : pour les mails, l'objet est écrit au dessus du texte et un bouton permet de le copier ;
  • Case à cocher : amélioration de certaine intéraction dans le formulaire d'écriture de texte avec des cases à cocher ;
  • Champs de texte plus grand : amélioration du champs de texte pour certaine demande de texte long.

Correction des messages types :

  • SMS de presciption : "Pole emploi" -> "France Travail".

Mise à jour mineur :

  • Dans la page d'accueil :
    • Bouton "Voir update" : permet d'afficher l'encart avec les dernières mise à jour ;
    • Bouton "Vider Localstorage" : permet de vider le Localstorage*.

*Les nouvelles fonctionnalités utilisent la sauvegarde locale permise par Localstorage pour conserver des informations lors de l'utilisation du site.
