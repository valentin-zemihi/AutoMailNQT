Information dernière mise à jour

02/06/2025 - Mise à jour V1.0

L'éditeur de message entre dans sa version 1.0 grâce à l'arrivée trois fonctionnalités :

  • L'édition des Textes Types ;
  • La sauvegarde des Textes Types localement ;
  • La sauvegarde et le chargement des Textes Types dans un fichier externe.

Ces nouvelles fonctionnalités permettent beaucoup plus d'autonomie de l'utilisateur dans l'usage du site internet, par la personnalisation des Textes Types et l'ajout de nouveau.

***Nouvelle fonctionnalité*** Editeur de message type

Une page "Editeur de texte type" est maintenant disponible !

Cette page permet d'éditer directement dans le navigateur les messages types.

Les Textes Types édités apparaissent en vert .

***Nouvelle fonctionnalité*** Sauvegarde de Texte Type

Sauvegarde local

  • Les Textes Types sont désormais sauvegardés dans votre navigateur via la fonctionnalité Localstorage.
  • Vous pouvez éditer un Texte Type via l'éditeur et le retrouver imédiatement à jour dans votre bibliothèque.
  • Attention, il est important de sauvegarder ces textes avant vider le Localstorage via le bouton dans l'accueil ou avant une mise à jour du site. Ces deux actions réinitialisent le Localstorage avec les valeurs par défauts.

Sauvegarde dans un fichier

  • Les Textes Types peuvent être désormais sauvegardés dans des fichiers .txt que vous pouvez enregistrer localement sur votre ordinateur.
  • Le site charge automatiquement les Textes Types par défaut, mais vous pouvez aussi importer un fichier .txt que vous avez précédemment sauvegardé.
  • Due à cette refonte de sauvegarde, la manière de manipuler les Textes Types ont aussi changer. Le changement est principalement dans le programme et non dans l'affichage, mais il peut y avoir des anomalies.

⚠️Important - Mes à jour complète des des Textes Types

Les Textes Types ne sont pas tous à jour et peuvent comporter des erreurs.

Version des Textes Types

  • Chaque Texte Type comporte désormais une version de mise à jour, ce qui permet de suivre facilement son évolution.
  • Les Textes Types non à jour par rapport à la version actuelle apparaissent en rouge : ils restent accessibles, mais peuvent comporter erreurs.

Mise à jour des KeyElem

Les parties du textes ouvert à l'édition sont identifiables par des mots clés, appelé KeyElem. La nouvelle fonctionnalité de sauvegarde et d'édition de texte ont conduit à des mise à jours de ses KeyElem.

  • Les KeyElem pour faire de la mise en page fonctionne desormais comme les balise HTML avec un startTag et un commun : exemple startStrong et endStrong pour <strong> </strong> ;

04/03/2024 - Mise à jour Beta V0.6

Pour cette version 0.6, l'expérience utilisateur a été complètement améliorée avec l'ajout de nouvelles fonctionnalités : page d'accueil avec les dernières informations, amélioration des éléments utilisateur, organisation des textes types, amélioration de la page éditeur.


La liste des principales nouvelles fonctionnalités :

    Page d'accueil
  • Encart mise à jour : informe les modifications apportées au site internet ;
  • Ajout d'une page d'accueil : permet de sélectioner ou ajouter un profil utilisateur avant de passer à la liste de message type.

    • Page bibliothèque
  • Ajout en-tête utilisateur : zone au dessus de la page où est affiché l'utilisateur actif, un bouton "Retour pour retourner à la page précédente et un bouton pour changer d'utilisateur ;
  • Classement des textes : les textes types ont été classés dans différents dossiers et sous-dossiers. Chaque dossier est associé à un public et un média, ou à un événement. Les sous-dossiers sont associés à une raison ;
  • Filtre par région : les textes affichés sont filtrés selon la région de l'utilisateur.

    • Page éditeur
  • Ajout d'un menu sur le côté : permet de changer de texte depuis l'éditeur ;
  • Ajout d'objet de mail : pour les mails, l'objet est écrit au dessus du texte et un bouton permet de le copier ;
  • Case à cocher : amélioration de certaine intéraction dans le formulaire d'écriture de texte avec des cases à cocher ;
  • Champs de texte plus grand : amélioration du champs de texte pour certaine demande de texte long.

Correction des messages types :

  • SMS de presciption : "Pole emploi" -> "France Travail".

Mise à jour mineur :

  • Dans la page d'accueil :
    • Bouton "Voir update" : permet d'afficher l'encart avec les dernières mise à jour ;
    • Bouton "Vider Localstorage" : permet de vider le Localstorage*.

*Les nouvelles fonctionnalités utilisent la sauvegarde locale permise par Localstorage pour conserver des informations lors de l'utilisation du site.
